# 🌾 AgriPredict Pro Max: AI-Powered Smart Farming Assistant

AgriPredict Pro Max is an advanced, AI-driven farming assistant developed as a final project for the **RYSERA STEM AI Course**. It aims to help Sri Lankan farmers mitigate climate risks and optimize crop yields using cutting-edge Generative AI.

## 🚀 Key Features
* **Strategic Climate Forecasting:** Provides highly technical 3-month (April, May, June) forecasts based on regional climate data.
* **Trilingual Support:** Accessible in **Sinhala, Tamil, and English** to cater to all local farming communities.
* **AI Voice Assistant (TTS):** Converts complex technical advice into clear audio for better accessibility.
* **Expert AI Advisor:** A real-time Q&A system for specific agricultural challenges.
* **Computer Vision Scanner:** (Beta) Capable of identifying plant diseases from leaf images.

## 🛠️ Tech Stack
* **Language:** Python
* **Frontend:** Streamlit
* **AI Engine:** Google Gemini AI (1.5 Flash / Pro)
* **Voice Engine:** gTTS (Google Text-to-Speech)
* **Deployment Tools:** Google Colab, Serveo
